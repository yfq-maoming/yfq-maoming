# 学校web课设，是通过校企合作的形式进行的，该项目在我的尚品汇项目之后，也算是比较轻松。项目采用的技术框架是vue，以及mockjs模拟数据便于进行展示，实现简单的前端交互。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
